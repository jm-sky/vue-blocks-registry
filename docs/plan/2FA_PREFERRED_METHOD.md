# 2FA preffered method

User should be able to choose the default, preffered method for 2fa, but still have an option to choose other one if configured.

After user login with email & password, and 2fa is required, the preffered method should appear.

## Github example on settings page

### Preferred 2FA method
Set your preferred method to use for two-factor authentication when signing into GitHub.  
`[Select input]` `(info icon)` Passkeys should be used primarily for standalone, password-less authentication.
